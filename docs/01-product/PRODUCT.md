# WearBrowser 产品说明文档

## 1. 产品概述

**WearBrowser** 是一款专门为 Android 手表与 Wear OS 设备设计的现代浏览器。

它不是手机浏览器的缩小版，而是围绕手表场景、单手操作、圆形屏幕、短时间浏览、低精度触控重新设计的浏览器产品。

产品目标：

> 在 1.x 英寸屏幕上提供自然、高效、舒适的网页浏览体验。

## 2. 目标用户

- 使用 Wear OS 或安卓手表的高级用户。
- 需要临时查看网页、文档、后台、监控页、论坛、搜索结果的用户。
- 关注隐私、开源、轻量体验的用户。
- 喜欢自定义和二次开发的开发者。

## 3. 核心场景

- 快速搜索问题。
- 查看 GitHub、论坛、Wiki、文档。
- 打开 Home Assistant、NAS、路由器、监控后台。
- 查看验证码页面、状态页面、短链接。
- 阅读短文、新闻、说明文档。

## 4. 核心特色

### 4.1 全局侧边返回

任何页面均支持左侧边缘滑动返回。

- WebView 历史优先。
- 无历史时回到主页或退出。
- 手势跟手。
- 可取消。
- 支持震动反馈。

### 4.2 全局右侧前进

右侧边缘滑动触发 Forward。

- 有历史时前进。
- 无历史时轻震提示。
- 后续支持长按 Forward History。

### 4.3 50%～300% 无极缩放

缩放范围：`50% ~ 300%`。

缩放精度：`1%`。

支持：

- 双指 Pinch。
- 单指边缘缩放。
- 菜单 +/-。
- 双击恢复 100%。
- Fit Width。
- 每站点记忆缩放。

### 4.4 单指边缘缩放

右侧边缘上下滑控制缩放。

- 上滑放大。
- 下滑缩小。
- 实时浮层显示当前倍率。
- 后续支持惯性和吸附点。

### 4.5 Edge Pie Menu

边缘长按弹出 Pie Menu。

主要动作：

- 返回。
- 前进。
- 搜索。
- 收藏。
- 刷新。
- Reader。
- 菜单。
- 设置。

### 4.6 圆形表盘优化

通过 Dynamic Safe Area 让网页核心内容不被圆形裁角遮挡。

支持模式：

- Auto。
- Strict。
- Reader。
- Fullscreen。

### 4.7 Watch Layout Engine

Watch Layout Engine 是项目的核心创新方向。

它不是简单缩放网页，而是分析页面结构，识别正文、导航、广告、Sidebar、Footer、评论等区域，并针对手表屏幕重排。

目标：打开即舒服，而不是让用户反复缩放。

## 5. 产品原则

- 默认隐藏浏览器 UI，把屏幕留给网页。
- 所有关键动作尽量可以单手完成。
- 圆屏不是兼容目标，而是一等公民。
- 设计系统统一控制颜色、间距、圆角、动画。
- 性能是功能的一部分。
- 默认本地保存，不上传用户浏览数据。

## 6. 功能总览

- WebView 浏览。
- 地址栏与搜索。
- 返回、前进、刷新、主页。
- 收藏夹。
- 历史记录。
- 标签页。
- Tab Sleep。
- 下载管理。
- Reader Mode。
- OLED Black。
- User Agent 切换。
- Site Profile。
- 手势引擎。
- 圆屏 Safe Area。
- Watch Layout Engine。
- Feature Flags。
- 工程化质量门禁。

## 7. 产品愿景

WearBrowser 希望成为世界上第一款真正为智能手表重新设计的开源浏览器。

更长期的目标是建立一套 Watch-first Web Interaction 设计范式，让手表网页浏览不再只是“能用”，而是自然、优雅、高性能。
