# Design Principles

## Watch First

所有设计先考虑手表，而不是迁移手机浏览器。

## Gesture First

关键动作应优先通过手势完成。

## Content First

网页内容是主角，浏览器 UI 默认隐藏。

## Round Screen First

圆屏是核心目标，不是边缘兼容场景。

## Minimal UI, Maximum Content

减少固定 UI，最大化网页显示面积。

## Performance is a Feature

流畅度、低内存、低功耗是核心体验。

## Privacy by Default

浏览记录、收藏、站点配置默认只保存在本地。

## Architecture Before Features

先确保抽象正确，再添加功能。

## Consistency Over Cleverness

统一设计系统优先于局部炫技。
