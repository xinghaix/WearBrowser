# Product Vision

WearBrowser 的长期愿景是：

> 建立智能手表上的 Watch-first Web Interaction 范式。

大多数手表浏览器只是将手机浏览器缩小，WearBrowser 的目标是重新设计网页浏览的输入、布局、显示、性能与交互。

## 核心信念

- 手表不是小手机。
- 手势优先于按钮。
- 内容优先于浏览器外壳。
- 圆屏必须被认真对待。
- 每一个像素都很珍贵。
- 性能本身就是产品体验。
- 开源项目也必须具备产品级质量。

## 长期目标

- 成为 Wear OS 上最适合实际使用的浏览器。
- 成为小屏 Web 布局优化的参考实现。
- 成为开发者研究手表交互的开源范例。
