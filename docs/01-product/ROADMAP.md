# Product Roadmap

## Phase 0：Foundation

- Clean Architecture。
- BrowserEngine 抽象。
- GestureEngine 抽象。
- Design System。
- Feature Flags。
- Repository 分层。

## Phase 1：Core Browser

- WebView 浏览。
- 地址栏与搜索。
- 返回 / 前进 / 刷新。
- 主页。
- 基础设置。

## Phase 2：Watch-first Interaction

- Edge Back。
- Edge Forward。
- Edge Zoom。
- Double Tap Reset。
- Pie Menu。
- Immersive Dock。

## Phase 3：Performance

- Tab Manager。
- Tab Sleep。
- Snapshot Cache。
- Memory Budget。
- Lazy Reader。
- Async Persistence。

## Phase 4：Watch Layout Engine

- DOM Block Detection。
- Content Classification。
- Importance Score。
- Round Safe Layout。
- Reader Integration。

## Phase 5：Product Polish

- 首次启动引导。
- 错误页。
- 无网络页。
- 设置页分组。
- 多语言。
- 图标与截图。

## Phase 6：Release

- Android 编译验证。
- APK Release。
- F-Droid 元数据。
- 隐私政策。
- 发布 Checklist。

## Phase 7：v2 Future

- 插件系统。
- 多设备同步。
- 自定义脚本。
- 更强的布局优化规则。
