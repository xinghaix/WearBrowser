# Security Policy

WearBrowser is a WebView-based browser. Please report security issues privately before opening public issues.

## Supported Version

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
| 0.x     | No        |

## Areas of Interest

- URL handling
- JavaScript settings
- File downloads
- Cleartext traffic
- Cookie/session handling
- WebView permission prompts
- Local data cleanup
